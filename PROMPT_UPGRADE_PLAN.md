# PROMPT_UPGRADE_PLAN: V10.8 → V10.9

## 목표
- QA에서 발견된 반복 오류를 프롬프트 레벨에서 차단
- 토큰 비용 절감 (중복 제거, 예시 축소)

## 변경사항 (우선순위순)

| # | 항목 | 현재 문제 | 변경 내용 |
|---|------|----------|----------|
| P1 | 비종목 필터링 | 암호화폐/원자재/통화가 종목으로 추출 (90건) | 명시적 제외 목록 추가, 비상장기업 시그널 제외 |
| P2 | key_quote 품질 | 자동자막 오류 텍스트 유입 (3건 심각) | 20~200자, 종목명 필수, 자막오류 감지 규칙 |
| P3 | signal_date | 오늘 날짜가 들어감 (9건) | 출력 필드에서 제거 (파이프라인에서 주입) |
| P4 | 중복 방지 | 같은 종목 2+ 시그널 (3건) | 기존 규칙 강화 (이미 있으나 미준수) |
| P5 | ticker 정확도 | 외국종목 ticker 누락 | ticker 모르면 null, 제외하지 말 것 |
| P6 | 매크로/교육 필터 | 매크로 영상에서 예시 종목 추출 | 투자 의견 vs 단순 예시 구분 규칙 |
| P7 | confidence 기준 | 기준 불명확 | 7 미만은 시그널 제외 |

## 정리 작업
- 0단계 영상 제목 필터링 → 프롬프트에서 제거 (코드 레벨 필터링)
- `signal` vs `signal_type` 필드명 통일 → `signal_type`만
- 중복된 검증 포인트/절대 금지 섹션 통합
- 예시 JSON 1개로 축소

## 예상 효과
- 토큰: ~2800 → ~1800 (약 35% 절감)
- 비종목 시그널 제거
- key_quote 품질 향상
- signal_date 오류 원천 차단
