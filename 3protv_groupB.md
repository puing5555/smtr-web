- ⏳ V4 파이프라인 분석: 자막 추출 완료 후 진행 예정

## 기술적 이슈 및 해결 방안
### 현재 문제
- YouTube에서 HTTP 429 (Too Many Requests) 에러 지속 발생
- 자막 추출 프로세스가 속도 제한에 걸림

### 적용 중인 해결 방안
1. **지수적 백오프 (Exponential Backoff)**: 재시도 간격을 점진적으로 증가
2. **사용자 에이전트 변경**: 브라우저 식별자 사용
3. **개별 다운로드**: 배치 다운로드 대신 영상별 개별 처리
4. **긴 대기 시간**: 2-5분 간격으로 재시도

### 현재 진행 상황
- 첫 번째 영상 (hxpOT8n_ICw) 자막 추출 시도 중
- 백그라운드 프로세스로 retry 로직 실행 중
- 성공 시 나머지 5개 영상 순차 처리 예정
