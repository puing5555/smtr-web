# VIDEO_FILTER_RULES.md - 영상 필터링 규칙

_Last updated: 2026-03-04_

모든 새 채널 영상 추출 시 이 규칙을 적용한다.

## ■ 추출 대상 (O)

### 1. 제목에 특정 종목명/티커가 있는 영상
- "엔비디아 실적 분석" → O
- "테슬라 지금 사도 될까" → O
- "삼성전자 HBM4 수혜" → O

### 2. 제목에 매수/매도/긍정/부정 시그널이 있는 영상
- "지금 반드시 사야 할 주식 3가지" → O
- "이 종목 팔아야 합니다" → O
- "올해 10배 갈 종목" → O

### 3. 제목에 실적/목표가/밸류에이션 키워드
- "4분기 실적 서프라이즈" → O
- "목표가 상향" → O

## ■ 제외 대상 (X)

1. **매크로/시황만:** "오늘 시장 정리", "미국 금리 전망"
2. **교육/입문:** "주식 시작하는 법", "PER이란"
3. **라이브 방송:** "실시간 장중 방송", "마감 시황 라이브"
4. **광고/멤버십:** "멤버십 전용", "유료 리포트"
5. **뉴스 단순 전달:** "오늘 뉴스 정리"

## ■ 판단 애매한 경우

"반도체 슈퍼사이클 온다" 같은 경우:
→ 자막 첫 5분 확인해서 개별 종목 분석 있으면 O, 없으면 X

## ■ 3단계 파이프라인

1. **1단계: 제목 필터링 (Haiku)** — 전체 영상 제목 배치로 O/X 판정
2. **2단계: 자막 추출 (yt-dlp)** — O인 영상만
3. **3단계: 시그널 추출 (Sonnet V10.10)** — 자막 → 시그널

## ■ 적용 범위

- 모든 새 채널에 동일 적용
- 기존 채널 재크롤링 시에도 적용
