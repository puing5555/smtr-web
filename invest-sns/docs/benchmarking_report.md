## 🔍 카테고리별 분석

## 1️⃣ 데이터/차트 분야

### 🏆 Finviz (finviz.com)
**핵심 기능:**
- 실시간 주식 스크리너
- 히트맵 시각화
- 기술적 지표 분석
- 뉴스 헤드라인 통합

**벤치마킹할 점:**
1. **실시간 데이터 대시보드**: 상승/하락, 신고가/신저가, RSI 지표를 한눈에
2. **시각적 정보 전달**: 테이블보다 차트와 히트맵으로 직관적 표현
3. **카테고리별 필터**: Top Gainers, New High, Unusual Volume 등

**우리 사이트 적용:**
- 유튜버 픽 종목들의 실시간 성과를 히트맵으로 표시
- 공시 발생 종목들을 카테고리별로 필터링 (긍정적 공시, 부정적 공시)
- 뉴스와 공시를 시간순으로 통합 표시

### 🏅 Koyfin (koyfin.com) 
**핵심 기능:**
- 커스터마이징 가능한 대시보드
- 10만+ 글로벌 종목 데이터
- 전문가급 차트 도구

**벤치마킹할 점:**
1. **완전 커스터마이징 UI**: 사용자가 원하는 대로 레이아웃 조정
2. **드래그 앤 드롭 방식**: 종목을 사이드바에서 끌어와서 분석
3. **테마 선택**: 다크/라이트 모드 등

**우리 사이트 적용:**
- 사용자별 관심종목 대시보드 커스터마이징
- 드래그 앤 드롭으로 유튜버 채널과 종목 연결
- 개인화된 알림 설정

## 2️⃣ 포트폴리오 관리 분야

### 🏆 Simply Wall St (simplywall.st)
**핵심 기능:**
- 포트폴리오 통합 관리 (2000+ 브로커 연동)
- "Snowflake" 시각적 분석
- 배당 품질 예측 시스템
- 일일 알림 시스템

**벤치마킹할 점:**
1. **포트폴리오 스노우플레이크**: 한 눈에 보는 포트폴리오 건강도
2. **배당 품질 스트레스 테스트**: 배당금 삭감 위험 사전 경고
3. **일일 맞춤 알림**: earnings, dividends, 밸류에이션 변화 알림
4. **브로커 통합**: 여러 계좌를 하나로 통합 관리

**우리 사이트 적용:**
- 유튜버가 추천한 종목들의 포트폴리오 성과 시각화
- 공시 기반 리스크 알림 (어닝스 쇼크, 매출 감소 등)
- 국내 증권사 API 연동으로 실제 보유종목과 유튜버 픽 매칭

## 3️⃣ 소셜/커뮤니티 분야

### 🏆 WallStreetBets (reddit.com/r/wallstreetbets)
**핵심 기능:**
- 400만+ 회원 투자 커뮤니티
- YOLO 포스트 (1만달러 이상 거래)
- Gain/Loss 인증 (2.5K 이상)
- 실시간 채팅 (Discord 연동)

**벤치마킹할 점:**
1. **증거 기반 검증**: 실제 거래 내역 스크린샷 필수
2. **최소 금액 제한**: 질 낮은 글 필터링 ($10K+ YOLO, $2.5K+ Gain/Loss)
3. **실시간 소통**: Discord 연동으로 실시간 채팅
4. **규칙 기반 운영**: 시총 5억불 이상만 허용, 정치 얘기 금지 등

**우리 사이트 적용:**
- 유튜버 픽의 실제 수익률 인증 시스템
- 최소 투자금액 기준으로 리뷰 품질 관리
- 실시간 종목 토론 채팅룸
- 펌프앤덤프 방지를 위한 시총/거래량 필터

## 4️⃣ 뉴스/정보 분야

### 🏅 株探(Kabutan) - 일본
**핵심 기능:**
- 실시간 결산속보
- 인기 뉴스 랭킹 (8시간 기준)
- 테마주 추적 (방위, 반도체, AI 등)
- 레이팅 일보 시스템

**벤치마킹할 점:**
1. **실시간 랭킹**: 인기 뉴스, 인기 종목, 검색어 순위
2. **테마별 분류**: 시장 이슈에 따른 테마주 자동 분류
3. **레이팅 변화 추적**: 증권사 목표가 변경사항 일일 업데이트
4. **PTS 야간거래 정보**: 정규시간 외 거래 현황

**우리 사이트 적용:**
- 유튜버 언급 빈도 기반 종목 랭킹
- 테마별 유튜버 픽 분류 (AI, 2차전지, 바이오 등)
- 애널리스트 리포트 vs 유튜버 의견 비교
- 시간외 거래에서 유튜버 픽 종목 추적

## 🎯 우리 사이트 차별화 전략

### 1. 속도 중심의 정보 전달
**기존 사이트들의 한계:**
- Seeking Alpha: 리포트 작성에 시간 소요
- Simply Wall St: 포트폴리오 분석 중심, 실시간성 부족
- Finviz: 일반적인 시장 데이터, 개인화 부족

**우리의 해결책:**
- **유튜버 영상 → 자동 분석 → 즉시 알림** 파이프라인
- 공시 발생 5분 내 관련 유튜버 과거 언급 내역 표시
- 실시간 채팅으로 유튜버 라이브 중 언급 종목 즉시 공유

### 2. 한국 특화 기능
**해외 사이트에는 없는 것들:**
- 국내 유튜버 채널 통합 모니터링
- 한국 공시시스템(DART) 실시간 연동
- 국내 증권사 MTS 연동
- 한국어 자연어 처리로 유튜버 발언 분석

### 3. 검증된 인사이트
**WallStreetBets의 검증 시스템 응용:**
- 유튜버 픽의 실제 수익률 추적 및 순위
- 허위 추천 필터링 (펌프앤덤프 방지)
- 커뮤니티 기반 팩트체킹

## 🚀 결론

해외 사이트들은 각각 특화된 영역에서 강점을 보이지만, **한국 유튜버 + 국내 시장 + 실시간 속도**를 결합한 사이트는 존재하지 않습니다.

**핵심 성공 전략:**
1. **Finviz의 실시간 시각화** + **Simply Wall St의 알림 시스템** + **WallStreetBets의 검증 문화**
2. 한국 특화 기능으로 차별화
3. 속도를 최우선으로 하는 UX 설계

이를 통해 **"투자 정보의 넷플릭스"**라는 포지셔닝으로 시장을 선점할 수 있을 것입니다.
