## 문서 1: 142개 채널 우선순위 기준

### 선정 기준 (가중치)

| 기준 | 가중치 | 설명 |
|------|--------|------|
| 구독자 수 | 25% | 10만↑ 우선, 영향력 지표 |
| 시그널 추출 적합도 | 25% | 개별 종목 분석 비율 (매크로/교육 채널은 감점) |
| 영상 빈도 | 15% | 주 2회↑ 활발한 채널 우선 |
| 카테고리 균형 | 15% | 한국주식/미국주식/크립토 균형 배분 |
| 자막 품질 | 10% | 자동자막 vs 직접자막, 한국어 정확도 |
| 중복도 | 10% | 기존 8개 채널과 겹치는 종목 커버리지 적으면 가점 |

### 카테고리별 목표 배분 (1차 20개)

| 카테고리 | 목표 | 비고 |
|----------|------|------|
| 한국주식 | 6개 | 기존에 삼프로/세상학개론 있으므로 차별화 필요 |
| 미국주식 | 8개 | 기존에 이효석/월가아재 있으므로 다른 관점 |
| 크립토 | 3개 | 새 카테고리, 수요 높음 |
| 복합 | 3개 | 한국+미국 모두 다루는 채널 |

### 1차 크롤링 대상 선정 프로세스

1. Research가 142개 채널에 위 기준으로 점수 매김
2. 상위 20개 리스트 JAY 확인
3. 확정된 채널부터 자막 크롤링 시작
4. V10.10 프롬프트로 시그널 추출
5. QA 2단계 검증 후 DB INSERT

## 문서 3: 우선순위 정리 (공시 vs 아이디어 수집)

### 비교

| 항목 | 공시 탭 | 투자 아이디어 수집 |
|------|---------|-------------------|
| 사용자 가치 | ⭐⭐⭐⭐⭐ 핵심 기능 | ⭐⭐⭐⭐ 차별화 기능 |
| 구현 난이도 | 중간 (~11시간) | 높음 (4-6주) |
| 데이터 소스 | DART API (안정적) | 네이버/유튜브/텔레그램 (복잡) |
| 비용 | 무료 (DART API) | 월 $324 |
| 법적 리스크 | 없음 (공개 API) | 있음 (크롤링) |
| 경쟁 우위 | 공시 + AI 분석 = 차별화 | 한국 웹 전체 수집 = 독보적 |

### 추천 순서

1. **공시 탭 먼저** (1-2일)
   - 핵심 기능, DART API 무료, 법적 리스크 없음
   - 시드 20건으로 UI 빠르게 만들고 JAY 피드백
   
2. **채널 확장** (1-2주)
   - 142개 중 20개 선정 → 크롤링 → 시그널 추출
   - 기존 파이프라인 활용, 추가 개발 최소

3. **투자 아이디어 수집** (2-6주)
   - Phase 1: 유튜브 + 텔레그램 (기존 인프라 활용)
   - Phase 2: 네이버 블로그 (검색 API)
   - Phase 3-4: 트위터, 카페, 커뮤니티
