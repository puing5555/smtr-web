# invest-sns 락인 전략 — AI 투자 비서 기획서 v1

---

## 핵심 컨셉

"내 포트폴리오 × 우리만의 데이터 = 나만의 AI 투자 비서"

- 사용자가 종목/매수가를 입력하면
- 우리가 쌓은 모든 데이터(유튜버 적중률 + 공시 AI분석 + 리포트 + 소셜 트렌드 + 수급)를
- **내 포트폴리오 기준으로 가공해서** 돌려줌
- → 같은 공시라도 내 매수가 기준으로 해석이 달라짐
- → 이건 다른 사이트에서 절대 못 해줌
- → 쓸수록 AI가 나를 알게 됨 → 떠날 수 없음

---

## 구현 우선순위

### Phase 1: 내 종목 대시보드 (2주)
- 종목 등록 + 매수가 입력
- 내 종목 기준 공시/유튜버/리포트 통합 표시
- 기본 알림
→ 락인 레벨 1~2

### Phase 2: 맞춤 AI 분석 (2주)
- 내 매수가 기준 공시 해석
- 유튜버 시그널 + 내 종목 연결
- "JAY님 기준" 분석 문구 생성
→ 락인 레벨 3~4

### Phase 3: 투자 기록 + AI 학습 (4주)
- 매매 기록 → 수익률 자동 계산
- "어떤 유튜버 따라갔을 때 돈 벌었는지"
- AI 패턴 분석 + 맞춤 추천
→ 락인 레벨 5~6

### Phase 4: 소셜/커뮤니티 (4주+)
- 포트폴리오 공개 (선택)
- 팔로우/팔로워
- 종목 토론
→ 락인 레벨 7
