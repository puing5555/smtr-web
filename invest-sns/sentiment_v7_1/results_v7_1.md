```
POST https://freesis.kofia.or.kr/meta/getMetaDataList.do
Content-Type: application/json

Body:
{
  "dmSearch": {
    "tmpV40": "9999",       // max rows
    "tmpV41": "1",
    "tmpV1": "D",           // D=일간, M=월간
    "tmpV45": "20150101",   // 시작일
    "tmpV46": "20260307",   // 종료일
    "OBJ_NM": "STATSCU0100000070BO"  // 서비스 ID
  }
}
```

| 서비스 ID | 내용 |
|----------|------|
| `STATSCU0100000060BO` | 증시자금추이 (투자자예탁금, 장내파생상품예수금 등) |

## 8. 기술 메모 (다음 세션 참고)

### KOFIA API 사용법
```python
import requests
session = requests.Session()
session.headers.update({
    'User-Agent': 'Mozilla/5.0...',
    'Content-Type': 'application/json;charset=UTF-8',
    'X-Requested-With': 'XMLHttpRequest',
    'Origin': 'https://freesis.kofia.or.kr',
    'Referer': 'https://freesis.kofia.or.kr/stat/FreeSIS.do?parentDivId=MSIS10000000000000&serviceId=STATSCU0100000070',
})
payload = {"dmSearch": {"tmpV40": "9999", "tmpV41": "1", "tmpV1": "D", 
           "tmpV45": "20150101", "tmpV46": "20260307", "OBJ_NM": "STATSCU0100000070BO"}}
r = session.post('https://freesis.kofia.or.kr/meta/getMetaDataList.do', json=payload, timeout=30)
data = r.json()['ds1']  # 2752 records
```

### KOFIA 데이터 단위 변환
- `TMPV2` 원시 단위: 만원 (10,000원)
- → 억원 변환: `value / 10000`
- → 조원 변환: `value / 10000000`
- 신용잔고 2026-03-05: 3,369,789,031 만원 = 33.7조원 ✅

### Naver fchart API (KOSPI, KOSDAQ 차트)
```python
url = f'https://fchart.stock.naver.com/sise.nhn?symbol=KOSPI&timeframe=day&count=300&requestType=0'
# 반환: XML, item data="YYYYMMDD|open|high|low|close|volume"
```
