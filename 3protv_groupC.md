## 문제점 및 해결 방안

### 현재 문제점
1. **yt-dlp HTTP 429 에러**: YouTube에서 자막 추출 요청을 차단
2. **JavaScript Runtime 부족**: yt-dlp가 JavaScript 엔진을 찾지 못함
3. **ffmpeg 부재**: 미디어 처리 도구 부족

### 해결 방안
1. **대기 후 재시도**: 몇 시간 후 다시 시도
2. **다른 도구 사용**: 
   - youtube-transcript-api (Python)
   - 브라우저 확장 프로그램
   - 수동 자막 복사
3. **환경 설정**:
   - Deno 설치 (JavaScript runtime)
   - ffmpeg 설치
