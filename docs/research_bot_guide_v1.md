# invest-sns 리서치봇 운영 가이드 v1

## 우리가 만드는 것
한국 투자자를 위한 주식 통합 사이트

**문제의식:** 투자자들이 봐야 하는 사이트가 너무 많다.
- 공시 → DART, KIND
- 증권사 리포트 → 각 증권사 HTS
- 유튜버 의견 → 유튜브 채널 10개+
- 뉴스 → 매경/한경/서경/연합
- 차트/데이터 → 네이버증권, 인베스팅닷컴
- 커뮤니티 → 종토넷, 디시, 텔레그램

→ **한곳에서 모아주고, AI가 의미 해석해주는 사이트**

**핵심 기능:** "내가 투자한 종목의 공시/유튜버 픽/보고서/뉴스를 가장 빠르게 본다"

## ■ 축 3: 투자자 현장 인사이트

### 모니터링 소스

| 소스 | 채널 | 체크 빈도 | 핵심 |
|---|---|---|---|
| 네이버 종토넷 | 인기 게시판 | 매일 | 개미 불만/니즈 |
| 디시 주식갤러리 | 실시간 | 주 3회 | 실시간 센티먼트 |
| 텔레그램 투자 채널 | AWAKE 등 | 매일 | 공시 반응 |
| 앱스토어 리뷰 | 증권앱 리뷰 | 주 1회 | 경쟁사 불만 = 우리 기회 |
| 유튜브 댓글 | 투자 채널 | 주 2회 | 시청자 질문 = 니즈 |

### 찾아야 하는 것
- **불만** — "XX 앱은 공시 알림이 너무 느려" → 우리 기회
- **니즈** — "유튜버가 말한 종목 수익률 추적하고 싶다" → 기능 아이디어
- **행동 패턴** — "나는 아침에 DART 먼저 보고 네이버증권 확인" → UX 설계 참고
- **과금 의향** — "이런 기능이면 월 1만원도 낼 의향" → 수익모델 검증
