# invest-sns 문체 가이드라인

_Last updated: 2026-03-05_

## 기본 원칙
- **반말 통일** (사용자 facing 텍스트)
- 예: "확인해보세요" ❌ → "확인해봐" ✅
- 단, 공식 안내/에러 메시지는 존댓말 OK ("데이터를 불러올 수 없습니다")

## 한글/영어 혼용 규칙
- **UI 라벨:** 한글 우선. 영어 병기 시 괄호 ("매수 (BUY)")
- **종목명:** 한글 표기 (삼성전자, SK하이닉스). 해외종목은 영문 OK (AAPL, TSLA)
- **시그널 타입:** 한글만 (매수/긍정/중립/경계/매도)
- **버튼:** 한글 ("영상 보기", "자세히 보기", "종목 분석")

## 버튼/라벨 통일
| 기능 | 통일 표기 | ❌ 금지 |
|---|---|---|
| 유튜브 링크 | 영상 보기 | 영상보기, 영상링크, Watch |
| 상세 페이지 | 자세히 보기 | 더보기, 상세보기, Detail |
| 종목 이동 | 종목 분석 | 종목보기, 종목페이지 |
| 공시 원문 | 공시 원문 | 공시보기, DART링크 |
| 리포트 원문 | 리포트 원문 | PDF보기, 리포트링크 |
| 닫기 | 닫기 | Close, X |
| 새로고침 | 다시 시도 | Retry, 새로고침 |

## AI 분석 톤
- **시그널 요약:** 객관적 서술. "~했다", "~전망이다"
- **공시 분석:** 팩트 중심. 수치 필수. "매출 XX억, 전년 대비 XX% 증가"
- **리포트 요약:** 핵심만. "XX증권, 목표가 XX만원 → XX만원 상향. 근거: ~"
- **금지:** "대박", "폭등", "폭락" 등 자극적 표현
- **금지:** "~것 같습니다", "~일 수 있습니다" 등 불필요한 헤징

## 숫자 표기
- 가격: 쉼표 구분 (123,456원 / $123.45)
- 퍼센트: 소수 1자리 (+12.3%, -5.7%)
- 날짜: YY.MM.DD (26.03.05)
- 시가총액: 억/조 단위 (시총 52.3조)

## 개발자 용어 금지 (사용자 facing)
| ❌ 금지 | ✅ 대체 |
|---|---|
| null, undefined | (표시 안 함 또는 "-") |
| error, exception | "일시적 오류" |
| loading | "불러오는 중..." |
| fetch, API | (노출 금지) |
| signal, payload | (노출 금지) |
| Supabase, DB | (노출 금지) |
| NaN, Infinity | (표시 안 함) |

## 띄어쓰기 주의
- "영상 보기" (O) vs "영상보기" (X)
- "자세히 보기" (O) vs "자세히보기" (X)
- "목표 가격" (O) vs "목표가격" (X) — 단, "목표가"는 한 단어로 OK
