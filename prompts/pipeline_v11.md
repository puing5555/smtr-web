- **본인 매매 경험**: 실제 매매했다고 밝히며 추천하는 경우

**예시**:
- "저는 SK하이닉스 30% 비중으로 늘렸습니다" → 매수 ✓
- "지금이 테슬라 진입 타이밍이에요" → 매수 ✓  
- "이 가격이면 삼성전자 사도 됩니다" → 매수 ✓

**🟢 긍정으로 분류해야 하는 표현:**
- **단순 전망**: "좋아 보인다", "전망이 밝다" (본인 매매 언급 없음)
- **관심 추천**: "관심 가져볼 만하다", "주목할 필요 있다"
- **장기 평가**: "장기적으로 유망하다" (구체적 시점/액션 불명확)
- **분석만 제공**: 종목을 분석만 하고 투자 액션 제안 없는 경우
- **타인 시각**: "시장에서 좋게 보고 있다", "애널리스트들이 긍정적"

**예시**:
- "엔비디아 전망이 밝아 보입니다" → 긍정 ✓
- "이 종목 관심 가져보시길" → 긍정 ✓
- "장기적으로는 유망한 기업" → 긍정 ✓

**부정 케이스 처리**:
- 전망 + 매수 권유 혼재 시 → 최종 메시지 기준
- "좋다 + 사라" → 매수 (액션 지시 우선)
- "전망 좋다" 단독 → 긍정 (전망만 제시)

- "~할 수밖에 없다" (필연성 전망)
- **V10.2 추가**: "앞으로 ~할 것", "~하게 될 것", "~하는 방향으로" (미래 전망)
- **V10.2 추가**: "시장에서는 ~", "애널리스트들은 ~" (타인 의견 전달)

**현상 설명/교육 (→ 시그널 제외)**:
- **V10.2 신규**: "~회사는", "~기업이", "이런 종목들이" (단순 현상 설명)
- **V10.2 신규**: "로봇 관련 주식이 올라가는 건" (현상 관찰)

**권유/추천 (→ 매수)**:
- ~해라/사라/담아라/들어가라/늘려라
- "지금 안 사면 후회한다" (간접적이지만 매수 권유)

- "분할매수/추가매수 하라" (구체적 매수 방법 지시)

- "극단적 상황에서는" = mention_type="시나리오", 시그널 제외
- "규제 통과하면" = confidence = very_low (규제 리스크)
- "~원 밑에서 매수/~원 위에서 매도" = mention_type="시나리오", 시그널 제외 (가격전략)

**V8 신규: 상황별 조건부**
- "20대는 ~, 50대는 ~" = mention_type="시나리오", 시그널 제외 (생애주기별)
- "은퇴 앞둔 분들은" = mention_type="시나리오", 시그널 제외 (특정 대상 한정)

**V11 신규: 가정형 발언 시그널 제외**
다음 패턴이 key_quote에 포함된 경우 시그널 생성 절대 금지:
- "만약 ~했다면", "~이었다면", "~이라면"
- "만약 내가 ~한다면", "가정하면", "가정하자면"
- "동면 상태에 들어가야 된다면", "10년 후라면"
- "만약 ~할 경우 ~했을 것이다" (완전 가상의 시나리오)

예시:
❌ "제가 10년 동안 동면 상태에 들어가야 된다면 저는 제 모든 돈을 비트코인에 넣었을 거예요."

❌ "만약 게임스톱이 장기 투자 대상이었다면 이럴 때 돈을 꾸준히 적었을 것"
  → 가정형: 현재 실제 매수 권유 아님 → 시그널 금지
✅ "저는 비트코인에 30% 비중 유지하고 있습니다." → 현재 사실 → 시그널 생성 가능

핵심 규칙 14: 바스켓/섹터 추천 처리 (V7 신규)

종목이 아닌 섹터/바스켓 단위 추천:
- "반도체 소부장 끝까지 갖고 가라" → market=SECTOR, signal=해당 강도
- "증권주 ETF 사라" → market=ETF, signal=해당 강도

종목 나열 + 바스켓 추천:
- 발언자가 여러 종목을 나열하며 "이런 바스켓으로 가라"

→ 개별 종목은 2분 이상 깊은 분석이 있는 경우에만 별도 시그널

강도 구분 기준:
- 깊은 분석 (2분+ 설명, 논거 제시, TP 언급) → 별도 시그널 (매수/긍정)
- 중간 분석 (30초~2분, 구체적 투자 논거 1개 이상) → 별도 시그널 (긍정)  ← V9.1 추가
- 간단 언급 (이름만, 1줄 설명) → 별도 시그널 없음. 바스켓에 포함

**V9.1 추가: 나열형 종목 누락 방지**
- 발언자가 3개 이상 종목을 순서대로 나열할 때, 뒤쪽 종목도 반드시 체크
- "첫 번째는 A, 두 번째는 B, 세 번째는 C" 패턴에서 C도 분석 대상
- 각 종목에 구체적 투자 논거(수혜 이유, 실적 전망 등)가 있으면 별도 시그널

**V9.1 추가: ETF 추천 누락 방지**
- V9 규칙4에서 ETF는 시그널 유지라고 명시했으나 실행 시 누락 빈발
- 발언자가 ETF를 직접 이름 언급 + 비중/배분 권유 → 반드시 시그널 생성
- 예: "SOXX ETF에 30% 투자하면 수익 낼 수 있다" → 긍정 시그널

예시:
- 박병창: "한미반도체, 유진테크, PSK — 이런 바스켓으로 가라"
  → 반도체 소부장 SECTOR: 긍정 (1개 시그널)
  → 개별 종목: 시그널 없음 (간단 나열)

- 조진표: "원익IPS — 삼성 설비투자 시 가장 먼저 발주 나오는 종목" (3분 설명)
  → 원익IPS: 매수 (깊은 분석으로 별도 시그널)
