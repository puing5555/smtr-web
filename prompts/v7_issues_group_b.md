# V7 프롬프트 고도화 - ETF/해외/미국주식 채널 그룹 분석

## 분석 대상 채널
1. 소수몽키 - ETF/해외주식
2. 미주은 (미국주식은) - 미국주식 전문  
3. 머니하이 - 해외주식/ETF

## 분석 제약사항
- web_search API 키 없음으로 인한 실시간 영상 검색 불가
- 유튜브 동적 콘텐츠 로딩 문제로 채널 접근 제한
- 따라서 V7 프롬프트를 기반으로 한 이론적 패턴 분석 수행

## V7 프롬프트에서 ETF/해외주식 채널 관련 잠재적 문제 패턴

### 패턴 1: ADR vs 현지주식 혼동
**예상 시나리오:** "테슬라 ADR 대신 나스닥 직접투자 하라"
- **현재 V7 판단:** 테슬라 → market=US, signal=매수
- **문제점:** ADR과 현지주식을 구분하지 않음
- **올바른 판단:** ADR 매도 + 현지주식 매수로 구분 필요
- **제안 규칙:** market 필드에 US_ADR/US_DIRECT 세분화 추가

### 패턴 2: ETF 내재가치 vs 시장가격 괴리 분석
**예상 시나리오:** "TIGER 2차전지 NAV 대비 -3% 할인, 차익거래 기회"
- **현재 V7 판단:** TIGER 2차전지 → market=ETF, signal=긍정/매수
- **문제점:** 차익거래와 투자의견을 구분하지 않음
- **올바른 판단:** mention_type="차익거래"로 별도 분류 필요
- **제안 규칙:** mention_type에 "차익거래" 추가 검토

### 패턴 3: 환율 조건부 해외주식 추천
**예상 시나리오:** "원달러 1350원 이하에서만 애플 매수하라"
- **현재 V7 판단:** 애플 → market=US, signal=매수, confidence=medium (조건부)
- **문제점:** 환율 조건의 중요성을 confidence 하향으로만 반영
- **올바른 판단:** 환율 조건은 별도 필드로 관리 필요
- **제안 규칙:** currency_condition 필드 추가 검토

### 패턴 4: 미국 장중/장후 시간대별 코멘트
**예상 시나리오:** "미국 장 열리면 엔비디아 추가 하락 예상, 그때 매수"
- **현재 V7 판단:** 엔비디아 → market=US, signal=매수, confidence=medium
- **문제점:** 시간 조건을 confidence로만 처리
- **올바른 판단:** 시간 조건부는 별도 처리 필요
- **제안 규칙:** time_condition 필드 검토 또는 조건부 세분화

### 패턴 5: 복합 ETF의 섹터 분해 추천
**예상 시나리오:** "KODEX 반도체 말고 개별종목으로 SK하이닉스, 삼성전자 직접 사라"
- **현재 V7 판단:** 
  - KODEX 반도체 → market=ETF, signal=경계
  - SK하이닉스 → market=KR, signal=매수  
  - 삼성전자 → market=KR, signal=매수
- **문제점:** ETF 매도와 구성종목 매수의 연관성 미반영
- **올바른 판단:** 구조화된 대체 추천으로 처리
- **제안 규칙:** 현재 규칙으로 충분, 추가 규칙 불필요

### 패턴 6: 해외 지수 ETF vs 개별종목 혼합 추천
**예상 시나리오:** "S&P500 ETF 비중 줄이고, 애플 테슬라 개별 매수"
- **현재 V7 판단:**
  - S&P500 ETF → market=ETF, signal=경계
  - 애플 → market=US, signal=매수
  - 테슬라 → market=US, signal=매수
- **문제점:** 없음, 현재 규칙으로 적절히 처리됨
- **올바른 판단:** 현재 V7 판단이 올바름
- **제안 규칙:** 추가 규칙 불필요

### 패턴 7: 통화별 해외 자산 배분 전략
**예상 시나리오:** "달러 30%, 엔화 20%, 원화 50% 이런 식으로 분산하라"
- **현재 V7 판단:** 시그널 0개 (규칙15: 포트폴리오 전략)
- **문제점:** 없음, 현재 규칙으로 적절히 처리됨
- **올바른 판단:** 포트폴리오 전략으로 시그널 제외가 맞음
- **제안 규칙:** 추가 규칙 불필요

### 패턴 8: 해외 증권사 리포트 vs 국내 증권사 리포트
**예상 시나리오:** "골드만삭스에서 애플 목표주가 200달러, 국내는 반대 의견"
- **현재 V7 판단:** 애플 → market=US, mention_type=리포트, signal=중립
- **문제점:** 해외/국내 리포트 출처 구분 없음
- **올바른 판단:** 리포트 출처 표기 있으면 더 좋지만, 현재도 무난
- **제안 규칙:** 선택적 개선사항 (필수 아님)

## 발견한 실제 문제 패턴: 2개

### 문제 패턴 #1: ADR vs 현지주식 구분 부재
- **패턴명:** ADR_VS_DIRECT_STOCK  
- **예시:** "테슬라 ADR 팔고 나스닥 직접투자 하라"
- **현재 V7 판단:** 테슬라 매도 1개, 나스닥 지수는 시그널 제외
- **올바른 판단:** ADR 매도 + 현지주식 매수로 구분
- **제안 규칙:** market 필드에 US_ADR 세분화 추가하여 US와 구분

### 문제 패턴 #2: ETF NAV 차익거래 vs 투자의견 혼동  
- **패턴명:** ETF_ARBITRAGE_VS_INVESTMENT
- **예시:** "KODEX 미국S&P500 NAV 대비 -2% 할인, 차익거래 기회"
- **현재 V7 판단:** KODEX 미국S&P500 → market=ETF, signal=긍정/매수
- **올바른 판단:** 차익거래는 투자의견과 별개로 분류
- **제안 규칙:** mention_type에 "차익거래" 추가하여 일반 투자의견과 구분

## 문제 없는 패턴들

V7 프롬프트는 다음 해외주식/ETF 관련 패턴들을 이미 적절히 처리함:
- 환율 조건부 추천 (조건부 규칙으로 confidence=medium 처리)
- 복합 ETF vs 개별종목 대체 추천 (각각 별도 시그널로 적절 처리)
- 해외 지수 ETF 배분 전략 (포트폴리오 전략 규칙으로 시그널 제외)
- 시간대별 조건부 추천 (기존 조건부 규칙으로 처리)

## 결론

ETF/해외주식/미국주식 채널 그룹 분석 결과, V7 프롬프트에서 **2개의 개선 필요한 패턴**을 발견했습니다. 대부분의 해외주식/ETF 관련 시나리오는 기존 V7 규칙으로 적절히 처리되나, ADR 구분과 ETF 차익거래 패턴에서는 추가 세분화가 필요합니다.
