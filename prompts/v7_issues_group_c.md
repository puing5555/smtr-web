## 1. 코인데스크코리아 (코인뉴스 채널)

### 채널 특성
- **유형**: 암호화폐 뉴스 전문 채널
- **콘텐츠**: 코인 시장 분석, 규제 뉴스, 기술적 분석
- **발언자**: 기자, 애널리스트, 게스트 전문가

### 발견된 잠재적 문제 패턴

#### 🚨 문제 1: 크립토 고유 용어 오분류
**문제**: V7 프롬프트가 크립토 특유의 표현을 잘못 분류할 수 있음

**예상 사례**:
- "비트코인 HODL 해라" → V7: 매수 (실제: 장기보유 권유, 긍정 정도)
- "알트시즌 온다" → V7: 긍정 (실제: 시장 전망, 중립)
- "디파이 토큰 디겐하지 마라" → V7: 경계 (크립토 속어 "디겐" 이해 필요)

**V7 규칙 부족**: 크립토 고유 용어(HODL, 알트시즌, 디겐, 에이핑 등)에 대한 처리 규칙 없음

#### 🚨 문제 2: 스테이킹/디파이 수익 구조 오분류
**문제**: 전통 주식과 다른 크립토 수익 구조를 잘못 해석

**예상 사례**:
- "이더리움 스테이킹하면 연 4% 받는다" → V7: 긍정 (실제: 교육/설명)
- "유니스왑 LP 제공하면..." → V7: 매수 (실제: 디파이 교육)

**V7 규칙 부족**: mention_type에 "디파이/스테이킹" 카테고리 없음

#### 🚨 문제 3: 규제 리스크 표현 오분류
**문제**: 규제 관련 조건부 발언의 복잡성

**예상 사례**:
- "규제 통과하면 리플 10달러 간다" → V7: confidence=medium (조건부)
- (실제: 규제 리스크는 binary 조건이므로 더 낮은 confidence 필요)

**V7 규칙 부족**: 규제 리스크 조건부 발언에 대한 별도 처리 규칙 없음

## 2. 부읽남TV (부동산+주식 혼합 채널)

### 채널 특성
- **유형**: 부동산 + 주식 통합 분석
- **콘텐츠**: 부동산 시장 전망, 부동산 관련주, 금리 영향 분석
- **발언자**: 부동산 전문가, 주식 애널리스트

### 발견된 잠재적 문제 패턴

#### 🚨 문제 4: 부동산-주식 연관 분석 오분류
**문제**: 부동산 시장 전망을 주식 시그널로 오분류 가능성

**예상 사례**:
- "집값 떨어지면 건설주도 같이 간다" → V7: 건설주 경계
- (실제: 부동산 전망이 주 내용, 건설주는 논거)
- "금리 인하되면 부동산 관련주 올라간다" → V7: 매수
- (실제: 조건부 + 섹터 전망, 긍정 정도)

**V7 규칙 부족**: 부동산-주식 연관성에서 주/부 분리 기준 모호

#### 🚨 문제 5: 지역별 부동산 vs 전국구 부동산주 혼재
**문제**: 지역 부동산 이야기와 부동산 관련 주식 구분 어려움

**예상 사례**:
- "강남 집값 오르면 GS건설 좋다" → V7: GS건설 매수
- (실제: 지역 부동산 분석이 주 내용, 개별주는 논거)

**V7 규칙 부족**: 지역 부동산 분석 시 관련주 처리 기준 없음

#### 🚨 문제 6: 금리-부동산-주식 3중 연관성
**문제**: 복합적 연관성에서 시그널 우선순위 모호

**예상 사례**:
- "금리 떨어지면 → 부동산 오르고 → 은행주는 안 좋다"
- V7 처리: 은행주 경계 (1개 시그널)
- (실제: 금리→부동산→은행의 3단계 논리, 각각 다른 confidence)

**V7 규칙 부족**: 다단계 인과관계 시그널 처리 규칙 없음

#### 🚨 문제 7: 절세/세무 전략과 투자 의견 혼재
**문제**: 세무 최적화 관점의 발언을 투자 추천으로 오분류

**예상 사례**:

- (실제: 세무 전략, mention_type에 "세무" 카테고리 필요)

- (실제: 절세 전략, 투자 의견 아님)

**V7 규칙 부족**: mention_type에 "세무/절세" 카테고리 없음

#### 🚨 문제 8: 자산 배분 vs 개별 종목 추천 혼재
**문제**: 포트폴리오 관점 vs 개별 종목 관점 구분 모호

**예상 사례**:
- "안전자산 20%, 성장주 30%, 배당주 50%" → V7: 시그널 없음 (전략)
- "근데 배당주 중에서는 SK텔레콤이..." → V7: SK텔레콤 긍정
- (실제: 배분 전략 내 예시, 별도 추천 아님)

**V7 규칙 부족**: 자산 배분 맥락에서의 종목 언급 처리 기준 모호

#### 🚨 문제 9: 생애주기별 투자 전략 오분류
**문제**: 연령/상황별 조건을 단순 조건부로 처리

**예상 사례**:
- "20대는 성장주, 50대는 배당주 위주로" → V7: confidence=medium (조건부)
- (실제: 일반론적 조언, 시그널로 분류하면 안 됨)
- "은퇴 앞둔 분들은 리츠 고려해보세요" → V7: REIT 긍정
- (실제: 특정 대상 한정, 일반적 투자 추천 아님)

**V7 규칙 부족**: 생애주기/상황별 조건부 발언 처리 규칙 없음
