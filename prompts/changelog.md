## V7.3 (2026-02-27)  
**타입**: 마이너 업데이트 - 뉴스 채널 및 복합 추천 규칙 추가

### 추가된 규칙 (뉴스 채널 + 복합 추천 대응)
1. **규칙 24 신규**: 뉴스 기자의 개인 의견 표현
   - "개인적으로는" + 투자 전망 → mention_type="결론", confidence="low"  
   - 뉴스 채널에서도 기자 개인 의견은 투자 조언 효과 있음
   - 단, 객관적 뉴스 전달과 구분하여 낮은 확신도 적용

2. **규칙 25 신규**: 복합 포트폴리오 내 핵심 종목 필터링
   - 포트폴리오 전략 + 다수 종목 나열 → "핵심" 표현 종목만 시그널
   - "~는 기본이고", "반드시", "핵심" 표현 → 시그널 생성
   - 단순 나열("A, B, C 정도?") → 바스켓으로 처리, 개별 시그널 없음

3. **규칙 4 확장**: market 분류 11종으로 확장  
   - 기존 10종 + REAL_ESTATE 추가
   - 부동산 지역/단지 → market=REAL_ESTATE
   - REITs → market=ETF (금융상품 우선)

### 테스트 채널
- 코인데스크 스타일: 뉴스 기자 개인 의견 처리 (2개 시그널)
- 부읽남TV 스타일: 복합 추천 필터링 (핵심 종목 3-4개로 축소)

   - 데이터 기반이지만 개인 의견 = 투자 조언으로 분류

3. **규칙 23 신규**: 과최적화 경고 동반 전략
   - 구체적 전략 제시 + 즉시 경고 → mention_type = "교육"
