# Pipeline 프롬프트 변경 이력

## V8.0 (2026-02-27)

### 📊 V8 업데이트 요약
- **분석 대상:** 22개 문제 패턴 (경제교양 9개 + ETF/해외 2개 + 코인/부동산/재테크 9개 + 퀀트/뉴스방송 4개)
- **통합 결과:** 6개 핵심 개선영역으로 통합
- **신규 규칙:** 4개 추가 (규칙 21-24)
- **기존 규칙 강화:** 5개 규칙 대폭 보강

### 🎯 핵심 개선사항

#### 1. mention_type 확장 (8종 → 11종)
```
V7: 결론/논거/뉴스/리포트/교육/티저/보유/컨센서스
V8: + 세무/차익거래/시나리오
```
- **세무:** 절세/세무 최적화 목적 종목 언급
- **차익거래:** ETF NAV 차익거래, 페어트레이딩
- **시나리오:** 복합조건부/시나리오 분석

#### 2. market 필드 세분화 (10종 → 13종)
```
V7: KR/US/JP/CN/CRYPTO/SECTOR/INDEX/ETF/MACRO/OTHER
V8: + US_ADR/CRYPTO_DEFI/REAL_ESTATE
```
- **US_ADR:** ADR과 현지주식 구분 (ETF/해외주식 채널용)
- **CRYPTO_DEFI:** 디파이 토큰 별도 분류 (코인 채널용)
- **REAL_ESTATE:** 리츠, 부동산 관련 상품

#### 3. confidence 등급 확장 (3단계 → 5단계)
```
V7: high/medium/low
V8: very_high/high/medium/low/very_low
```
- **very_low:** 규제 리스크 조건부 발언용

### 🆕 V8 신규 규칙 (4개)

#### 규칙 21: 세무/절세 전략 구분
**문제:** 신사임당 등 재테크 채널에서 절세 목적 발언을 투자 추천으로 오분류
**해결:** mention_type="세무"로 별도 분류, 투자 의견과 구분

#### 규칙 22: 퀀트/데이터 분석 구분  
**문제:** 강환국(할수있다알고투자) 등에서 데이터 해석과 투자 추천 경계 모호
**해결:** 데이터 나열=교육, 데이터+추천표현=결론으로 명확히 구분

#### 규칙 23: 부동산-주식 연관성 처리
**문제:** 부읽남TV 등에서 부동산 전망을 건설주 시그널로 오분류
**해결:** 부동산이 주 내용일 때 관련주는 mention_type="논거"로 처리

#### 규칙 24: 크립토 고유 표현 처리
**문제:** 코인데스크코리아 등에서 HODL, 알트시즌 등 용어 오분류
**해결:** 크립토 용어 정규화 + CRYPTO_DEFI 시장 추가

### 🔧 기존 규칙 대폭 강화 (5개)

#### 규칙 5: 교육/설명 구분 **대폭 강화**
**기존:** 기본적 교육 패턴만 처리
**V8:** 교육 패턴 8개 추가
- "역사적으로 ~했다" + 의문문 = 교육
- "차트상/기술적으로 ~가능성" = 분석 해설
- "이론상/원리상" = 경제 원리 설명
- "내재가치/이론가격" = 가치 분석 교육
- "리스크 관리 차원에서" = 교육
- "~전략에 넣으면 좋다" = 전략 설명

**해결 문제:** 슈카월드, 달란트투자 등 경제교양/가치투자 채널의 교육-의견 혼동

#### 규칙 6: 조건부 발언 **대폭 강화**
**기존:** 단순 조건부만 처리 (confidence=medium)
**V8:** 복합 조건부 처리 추가
- "만약 A하고 동시에 B한다면" = 시나리오 (시그널 제외)
- "극단적 상황" = 시나리오 (시그널 제외)
- "규제 통과하면" = very_low confidence
- "~원에서 매수/매도" = 가격 전략 (시그널 제외)
- 생애주기별 조건 = 시나리오 (시그널 제외)

**해결 문제:** 슈카월드 복합조건부, 신사임당 생애주기별 조건 오분류

#### 규칙 8: 게스트 채널 **보강**
**기존:** 기본적 게스트 구분
**V8:** 방송 형태별 특수 처리 추가
- 토크쇼 의견 충돌: 최종 발언 기준
- 뉴스 앵커 편향 감지: "반가운 소식" 등 → confidence=low

**해결 문제:** 장르만여의도 토크쇼, 주코노미 뉴스 채널 특수성

#### 규칙 10: 목표주가 **보강**
**기존:** 기본적 TP 처리
**V8:** 가치투자 구분 추가
- "내재가치/이론가격" = 교육 (가치 분석법)
- "목표주가" = 리포트 처리

**해결 문제:** 달란트투자 등 가치투자 채널의 내재가치와 목표주가 혼동

#### 규칙 17: 과거 보유 **보강**
**기존:** 과거 보유 기본 처리
**V8:** 장기보유 철학 추가
- "평생/계속 보유" = 보유 철학 (시그널 제외)
- "우량주는 평생 가져가야" = 투자 철학

**해결 문제:** 달란트투자 등 가치투자 채널의 보유 철학과 매수 추천 혼동

### 📈 교차검증 강화 (14항목 → 17항목)
