## 텔레그램 봇용 통합 프롬프트

```
아래 유튜브 채널의 영상을 분석해줘.
채널: {CHANNEL_URL}

⭐ **분석 스코프**: 한국주식 + 미국주식 + 크립토(코인)만 다룸
   - 부동산, 채권, 원자재(금/원유 등), 해외주식(일본/중국 등) 제외
   - 환율/금리는 시장 배경 언급만 가능, 직접 투자 시그널 생성 금지

⭐ **발언자 귀속**: 시그널은 채널이 아닌 발언자(speaker)에게 귀속
   - 같은 인물이 다른 채널 출연 시에도 동일 발언자로 추적
   - 발언자 이름 정규화가 매우 중요 (STEP 2.5 참고)

STEP 1.5: 제목 사전 필터링
- pass: 특정 종목 언급, 시장/섹터 전망, 매매 의견, 경제지표의 시장 영향
- skip: 구독자 Q&A, 일상/브이로그, 채널 공지, 투자 무관 교양/역사
⚠️ 제목은 필터링에만 사용. 시그널 판단에는 절대 사용 금지 (제목 과장 문제)

STEP 2: 통과한 영상 자막 추출 (한국어)

STEP 2.5: 자막 종목명 정규화
════════════════════════════════════
자막 추출 직후, 분석 전에 종목명 정규화 수행:
- 음성인식 오류 교정: 이너택→LG이노텍, 하이니스→SK하이닉스, 최영주→차영주
- 약칭→정식명: 삼전→삼성전자, 하닉→SK하이닉스, 현차→현대차
- 영문→한글 매칭: NVIDIA→엔비디아, TSLA→테슬라
- 인물명 정규화: 같은 소속+직함+코너명이면 동일 인물
  예: "박명성" = "박명석" = "글로벌 시황맨" → "박명성"으로 통일
- **V8 추가: 크립토 용어 정규화**: HODL→장기보유, 알트시즌→알트코인 시즌, 디겐→고위험투자

STEP 3: 전체 영상 상세 요약 (각 영상마다)
════════════════════════════════════
- 최소 500자 이상 상세 요약
- 영상 흐름 (시작 → 논거 전개 → 결론)
- 반론/반박 구조 명시
- 조건부 발언 명시 (조건 원문 포함)
- 확신 정도 (강하게 추천 vs 조심스럽게)
- ⭐ 영상의 최종 결론 명확히 기술
- ⭐ 각 종목의 언급 목적 분류 (11종): **V9 확장**
  - "결론" = 발언자 본인이 투자 의견을 제시하는 종목
  - "논거" = 다른 종목의 투자 논리를 뒷받침하기 위한 비교/예시
  - "뉴스" = 단순 시황/뉴스 전달 (본인 투자 의견 없음)
  - "리포트" = 증권사 리포트/목표주가 전달
  - "교육" = 현상 설명, 원리 해설, 역사적 맥락 (투자 의견 없음)
  - "티저" = 유료 콘텐츠/멤버십 전환을 위한 미끼 (V7 신규)
  - "보유" = 과거/현재 보유 경험 언급, 현재 매매 의견 아님 (V7 신규)
  - "컨센서스" = 복수 증권사 목표주가/의견 종합 전달 (V7 신규)
  - **V8 신규: "세무" = 절세/세무 최적화 관점의 종목 언급**
  - **V8 신규: "차익거래" = ETF NAV 차익거래, 페어트레이딩 등**
  - **V8 신규: "시나리오" = 복합조건부/시나리오 분석**

STEP 3.5: 발언자 추출
- 채널명 아닌 실제 발언자 이름
- 진행자와 게스트 구분
- 게스트 여러 명이면 각각 분리
- ⭐ 발언자 이름 정규화 적용 (STEP 2.5에서 정의)
- ⭐ **중요**: 시그널은 발언자에게 귀속됨 (채널 아님)
  - 동일 인물이 다른 채널 출연 시에도 같은 발언자로 추적
  - 박명성이 슈카월드/삼프로TV 출연 → 모두 "박명성" 시그널
  - 발언자 정규화를 통한 일관된 인물 추적이 핵심

핵심 규칙 1: 논거 vs 결론 구분
═══════════════════════
- "결론 종목": 발언자가 실제 투자 의견 제시 → 강도에 맞게 분류
- "논거 종목": 비교/예시로 사용 → 최대 "긍정"까지만. "매수" 분류 금지.
- "뉴스 종목": 시황 전달 → "중립"
- "리포트 종목": 증권사 리포트 전달 → 규칙2 참고
- "교육 종목": 설명/해설 → 시그널 제외 (테이블에서 제외)

예시:
"삼성전자 영업이익 209%↑! 코인 중에는 캔톤만 실적 있다"
→ 삼성전자: 논거 → 🟢긍정 (매수 아님)
→ 캔톤: 결론 → 🔵매수

═══════════════════════
핵심 규칙 2: 리포트 브리핑 구분
═══════════════════════
(a) 리포트 내용만 전달 (본인 의견 없음)
→ mention_type = "리포트", 신호 = "중립"
예: "교보증권에서 목표주가 37만원으로 상향했습니다" (해설만)

(b) 리포트 전달 + 본인 동의/추천 표현
→ mention_type = "결론", 신호 = 강도에 맞게
예: "이 종목 저는 강력 추천드립니다"

(c) 리포트 전달 + 본인 반대 표현
→ mention_type = "결론", 본인 의견 기준으로 분류

(d) 리포트 전달 + 미묘한 긍정 뉘앙스
→ mention_type = "리포트", 신호 = "중립", confidence = "low"
→ reasoning에 "미묘한 동의 뉘앙스 존재하나 명시적 추천 아님" 기록
예: "이수페타시스... 장기 투자자에게 솔깃한 뉴스"

판단 기준: "발언자가 본인 돈을 넣겠다는 뉘앙스가 있는가?"

═══════════════════════
핵심 규칙 3: 전망 vs 매수 권유 구분
═══════════════════════
동사 어미로 판단:

전망 (→ 긍정):
- ~이다/된다/보인다/전망이다/예상된다
- "코스피 7000 갈 수 있다"
- "충분히 가능하다고 봅니다"
- "올라갈 수밖에 없다" (강한 전망도 전망)
- "못 사면 어쩔 수 없다, 사면 좋겠다" (희망 표현)

권유 (→ 매수):
- ~해라/사라/담아라/들어가라/늘려라
- "지금 안 사면 후회한다" (간접적이지만 매수 권유)
- "~하시면 좋겠다/추천드립니다" (공손한 권유)
- "비중 없으면 지금이라도 들어가라"

핵심 규칙 4: 지수/섹터/ETF 분류 **스코프 제한**
═══════════════════════
market 필드 (9종): **V9 스코프 축소**
KR / US / US_ADR / CRYPTO / CRYPTO_DEFI / SECTOR / INDEX / ETF / OTHER

**⚠️ 스코프 제한: 한국주식 + 미국주식 + 크립토만 다룸**

- INDEX (코스피, S&P500, 나스닥 등):
  → 시그널 테이블에서 제외. 시장 전망으로만 요약에 기록.

- SECTOR (증권주, 건설주, 반도체 섹터 등):
  → 시그널 유지, market=SECTOR 표기
  예: "증권주 사라" → 🔵매수, market=SECTOR

- ETF (TIGER 반도체, KODEX 2차전지, SOSX 등):
  → 시그널 유지, market=ETF 표기
  → 구성종목으로 전파 금지 (ETF 시그널 ≠ 개별종목 시그널)

- **US_ADR** (테슬라 ADR, 애플 ADR 등) vs US (현지주식) 구분
- **CRYPTO_DEFI** (유니스왑, 컴파운드 등 디파이 토큰)

**⚠️ 스코프 밖 - 시그널 생성 금지:**
- **MACRO** (환율, 금리): 시장 배경 언급만 가능, 직접 투자 시그널 생성 금지
- **원자재** (금, 원유, 구리 등): 완전 제외
- **부동산** (리츠, 부동산펀드): 완전 제외  
- **해외주식** (일본, 중국, 유럽 등): 완전 제외
- 위 자산 언급 시 → 요약에만 기록, 시그널 테이블에서 제외

핵심 규칙 5: 교육/설명형 콘텐츠 구분 **V8 강화 + V7.3 유지**
═══════════════════════
mention_type = "교육" 판단 기준:
- "~인 이유는", "~의 구조는", "~을 알아보자" = 교육
- 현상 설명, 원리 해설, 역사적 맥락만 제공
- 교육 → 시그널 제외 (테이블에서 제외)

**V8 추가: 교육 패턴 확장**
- "역사적으로 ~했다" + "지금도 ~일까?" = 교육 (의문문 패턴)
- "차트상/기술적으로 ~가능성" = 분석 해설 → 교육
- "이론적으로/원리상 ~하다" = 경제 원리 설명 → 교육
- "내재가치/이론가격 계산하면" = 가치 분석 교육
- "리스크 관리 차원에서 손절라인 정해라" = 교육
- "~전략에 이런 종목들을 넣으면 좋다" = 전략 설명 → 교육

**V7.1 추가: 교육/투자의견 경계 케이스**
전망 제시 후 즉시 거리두기 표현 → 교육으로 분류:
- "HDC현대산업개발이 좋아지죠. 하지만 교과서적인 이야기고"
- "일반적으로 ~하지만 실제로는 다른 변수가 많습니다"
- 판단 기준: 발언자가 책임회피/거리두기 표현을 동반했는가?

단, 교육 후 "그래서 나는 ~한다" → 결론 (교육 아님)

═══════════════════════
핵심 규칙 6: 조건부 발언 처리 **V8 강화 + V7.3 유지**
═══════════════════════
**기존 단순 조건부:**
조건부 발언 감지 시:
- "~한다면", "~될 경우", "이 가격대에서는", "1분기까지는", "만약~"
→ confidence = medium (무조건 하향)
→ reasoning에 조건 원문 포함

**V8 신규: 복합 조건부**
- "만약 ~하고 동시에 ~한다면" = mention_type="시나리오", 시그널 제외
- "극단적 상황에서는" = mention_type="시나리오", 시그널 제외
- "규제 통과하면" = confidence = very_low (규제 리스크)
- "~원 밑에서 매수/~원 위에서 매도" = mention_type="시나리오", 시그널 제외 (가격전략)

**V8 신규: 상황별 조건부**
- "20대는 ~, 50대는 ~" = mention_type="시나리오", 시그널 제외 (생애주기별)
- "은퇴 앞둔 분들은" = mention_type="시나리오", 시그널 제외 (특정 대상 한정)

핵심 규칙 7: 기존 규칙 유지
═══════════════════════
- 영상 1개 × 종목 1개 × 발언자 1명 = 신호 1개
- 반론 소개 후 반박 → 최종 입장 기준
- "~던져라" = 명시적 매도
- "실적이 뭐예요?" (의문 제기) ≠ "팔아라" (매도 권유)

핵심 규칙 8: 게스트 채널 규칙 **V8 강화 + V7.3 유지**
═══════════════════════
- 진행자 질문은 신고 아님
- 게스트 답변만 신호로 분류
- 같은 영상에서 게스트 의견 다르면 각각 별도 신호

**V8 추가: 토크쇼 의견 충돌 처리**
- 게스트 간 실시간 반박으로 의견 변경 시 최종 발언 기준
- 진행자 "시청자들이 판단하시길" → 시그널 아님
- 불확실한 결론 시 confidence="low"

- 시황/뉴스 앵커: 기본값 = 뉴스/중립, 명시적 권유 표현 있을 때만 상향

**V8 추가: 뉴스 앵커 암시적 편향 감지**
- "반가운 소식", "우려되는 부분", "투자자들에게는 ~" 표현
→ mention_type="뉴스", signal="중립", confidence="low"
→ reasoning에 "미묘한 편향 감지" 기록

- 패널 토론에서 "저도 동의합니다"는 독립 시그널 아님

핵심 규칙 9: 중복 처리
═══════════════════════
같은 발언자가 여러 영상에서 같은 종목 언급:
- 발언자 이름 정규화 후 중복 판단
- 최신 영상 기준 1개만 남김
- 의견 변경 시 최신 채택 + reasoning에 "이전 의견: ~에서 변경됨" 기록

- 종목이 공개되지 않았으므로 시그널 생성 금지
- reasoning에 "유료 전환 티저: 종목 미공개" 기록
- ⚠️ 티저에서 암시된 섹터를 추론하여 종목 시그널을 만들지 말 것

**V7.1 추가: 티저 섹터 힌트 처리**
종목명은 숨기고 섹터만 공개하는 경우:
- "전기차배터리 관련, 반도체장비" + "자세한 건 유료에서"
→ 섹터 정보도 mention_type="티저"로 분류, 시그널 생성 금지
→ 힌트성 섹터 정보는 투자 의견이 아닌 유료 전환용 미끼

예시:
- 이효석: "반도체 다음 4종목 — 자세한 건 효친클럽에서"
  → 시그널 0개. 요약에만 "지주/전력기기/조선/로봇 섹터 언급 (티저)" 기록

핵심 규칙 14: 바스켓/섹터 추천 처리 (V7 신규)
═══════════════════════
종목이 아닌 섹터/바스켓 단위 추천:
- "반도체 소부장 끝까지 갖고 가라" → market=SECTOR, signal=해당 강도
- "증권주 ETF 사라" → market=ETF, signal=해당 강도

종목 나열 + 바스켓 추천:
- 발언자가 여러 종목을 나열하며 "이런 바스켓으로 가라"
→ 바스켓으로 1개 SECTOR 시그널 생성
→ 개별 종목은 2분 이상 깊은 분석이 있는 경우에만 별도 시그널

강도 구분 기준:
- 깊은 분석 (2분+ 설명, 논거 제시, TP 언급) → 별도 시그널 (매수/긍정)
- 중간 분석 (30초~2분, 구체적 투자 논거 1개 이상) → 별도 시그널 (긍정)  ← V9.1 추가
- 간단 언급 (이름만, 1줄 설명) → 별도 시그널 없음. 바스켓에 포함

**V9.1 추가: 나열형 종목 누락 방지**
- 발언자가 3개 이상 종목을 순서대로 나열할 때, 뒤쪽 종목도 반드시 체크
- "첫 번째는 A, 두 번째는 B, 세 번째는 C" 패턴에서 C도 분석 대상
- 각 종목에 구체적 투자 논거(수혜 이유, 실적 전망 등)가 있으면 별도 시그널

**V9.1 추가: ETF 추천 누락 방지**
- V9 규칙4에서 ETF는 시그널 유지라고 명시했으나 실행 시 누락 빈발
- 발언자가 ETF를 직접 이름 언급 + 비중/배분 권유 → 반드시 시그널 생성
- 예: "SOXX ETF에 30% 투자하면 수익 낼 수 있다" → 긍정 시그널

예시:
- 박병창: "한미반도체, 유진테크, PSK — 이런 바스켓으로 가라"
  → 반도체 소부장 SECTOR: 긍정 (1개 시그널)
  → 개별 종목: 시그널 없음 (간단 나열)

- 조진표: "원익IPS — 삼성 설비투자 시 가장 먼저 발주 나오는 종목" (3분 설명)
  → 원익IPS: 매수 (깊은 분석으로 별도 시그널)
```

```
V10 신규 규칙 29: 화자 식별 강화
═══════════════════════
자막에서 화자 구분이 안 될 때 추가 소스 활용:
1. **영상 제목 패턴**: "| 이름 직함" (예: "| 권영화 교수 1부") → 게스트 식별
2. **영상 설명란**: 게스트 정보, 출연자 소개 활용
3. **자막 내 호칭**: "교수님", "대표님", "박사님", "위원장님" → 게스트 발언 구간 식별
4. **대화 패턴**: "오늘 모신 분은", "~님 어떻게 보시나요?" → 게스트 전환점

⚠️ 채널 호스트를 게스트 발언의 speaker로 귀속시키지 말 것.
게스트가 있는 영상에서 호스트 이름이 speaker가 되면 오류.
확인 방법: 채널명 ≠ speaker일 때 → 정상 (게스트), 채널명 = speaker일 때 → 게스트 확인 필요

V10 신규 규칙 30: key_quote 품질 기준
═══════════════════════
핵심 발언(key_quote)은 반드시 다음 3가지를 충족:
1. **종목명 포함**: 어떤 종목에 대한 발언인지 명확
2. **투자 판단 근거 포함**: 왜 매수/매도/긍정인지 이유가 담긴 문장
3. **최소 15자 이상**: 맥락 없는 짧은 문장 금지

❌ 나쁜 예시:
- "둘 다 사는게 좋습니다" (종목명 없음, 근거 없음)
- "좋아보입니다" (근거 없음, 15자 미만)
- "사세요" (모든 조건 미충족)

✅ 좋은 예시:
- "SK하이닉스는 HBM4 수혜가 확실하고, 삼성 대비 기술 격차가 벌어지고 있어서 추가 매수합니다"
- "삼성전자는 반도체 업황 턴어라운드 시점이 다가오고 있어서 지금 분할 매수 구간"

전후 문맥을 포함해서 인용하되, 핵심 판단이 담긴 문장을 선택할 것.
```
