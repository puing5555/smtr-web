## 🔬 투자 SaaS 해부 #001 — TipRanks
📅 조사일: 2026-03-05
🏷 카테고리: [A] 종목 분석/데이터, [B] 뉴스/리포트
🌍 국가: 미국 (본사: 이스라엘 텔아비브)
🔗 URL: https://www.tipranks.com

### 1. 한줄 요약
월가 애널리스트·블로거·인사이더의 추천을 추적하고, 그들의 수익률을 기반으로 랭킹하여 개인투자자에게 '누구 말을 들어야 하는지' 알려주는 금융 데이터 플랫폼.

### 2. 타겟 고객
- **주 타겟:** 미국 개인투자자 (중급~고급)
- **부 타겟:** 증권사/은행 B2B (화이트라벨)
- **고객 규모:** 월간 방문자 약 3,000만+ (SimilarWeb 추정), B2B 파트너 10개+ (E-Trade, TD Ameritrade, eToro, Interactive Brokers, CIBC, Questrade, Futu, Nasdaq.com, Santander 등)

### 3. 핵심 기능 (최대 5개)
- **Smart Score:** 8가지 요소(애널리스트 컨센서스, 인사이더, 헤지펀드 등) 종합한 1~10점 AI 점수. 종목 빠르게 평가 가능
- **Analyst Rankings & Tracking:** 전체 월가 애널리스트의 과거 추천 수익률을 추적·랭킹. "이 애널리스트 말 믿어도 되나?" 정량적 답변
- **Top Analysts & Best Performing Experts:** 연도별 최고 성과 애널리스트 발표. 투자 의사결정의 신뢰 기반
- **Insider Trading Tracker:** 기업 내부자(경영진, 이사) 매매 내역 실시간 추적
- **Hedge Fund Portfolio Tracker:** 13F 기반 헤지펀드 포트폴리오 변화 추적

### 4. 수익모델
- **과금 방식:** 프리미엄 구독 (B2C) + 화이트라벨 API/위젯 (B2B)
- **가격대:**
  - Basic: 무료 (제한된 기능)
  - Premium: ~$29.95/월 (연 결제 시 할인, ~$199/년)
  - Ultimate: ~$49.95/월
- **B2B:** 증권사에 데이터 위젯/API 제공 (라이선스 수익, 금액 비공개)
- **ARPU 추정:** B2C $20~30/월, B2B가 매출의 상당 부분

### 5. 투자/성장
- **투자:** 2021년 4월 $77M 시리즈 라운드 발표 (Wikipedia 확인)
- **투자자:** Eliot Spitzer (전 뉴욕 주지사, 2014년 이사회 합류)
- **설립:** 2012년 (Uri Gruenbaum, Gilad Gat 공동 창업)
- **고객 수/MAU:** 월 3,000만+ 방문 추정 (미국 개인투자 리서치 사이트 중 최상위권)
- **성장:** 2012 → 2026, 14년간 미국/캐나다/영국/호주/독일 확장. 주요 증권사 10개+ 파트너십

### 6. 강점/약점
- 💪 **강점:**
  - 애널리스트 수익률 추적이라는 독보적 데이터 해자 (경쟁사가 따라하기 어려움)
  - B2B 파트너십 (E-Trade, Interactive Brokers 등)이 고객 유입 채널 + 안정 수익원
  - Smart Score 같은 종합 점수 → 초보도 쉽게 이해
  - 데이터 축적 기간 10년+ → 신규 진입자 대비 압도적 데이터 양
- 😤 **약점:**
  - 프리미엄 가격이 비쌈 ($30/월은 한국 기준으로 높음)
  - 미국 주식 중심 → 한국/아시아 커버리지 없음
  - 무료 플랜 기능 제한이 심해 "프리미엄 강매" 느낌 (앱스토어 리뷰 불만)
  - 커뮤니티/소셜 기능은 약함 (데이터 중심, 토론 없음)

### 7. invest-sns 적용 분석 ⭐️ (가장 중요)

#### 우리가 가져올 기능:
1. **유튜버 수익률 추적 시스템 (TipRanks 핵심 이식)**
   - TipRanks가 "애널리스트 추천 수익률 추적"하듯, 우리는 **"유튜버 픽 수익률 추적"**
   - 유튜버가 A종목 매수 추천한 날짜 기준 → 1주/1개월/3개월/6개월/1년 수익률 자동 계산
   - **유튜버 랭킹 시스템:** "이 유튜버 추천 종목의 평균 수익률은 +12.3%, 적중률 67%"
   - 이것이 우리 사이트의 **핵심 차별화 포인트**

2. **종목별 유튜버 컨센서스**
   - TipRanks의 "Analyst Consensus"처럼 → "유튜버 컨센서스" 구현
   - 삼성전자 페이지: "10명 중 7명 매수 의견, 평균 목표가 8만원"
   - 시각적으로 매수/보유/매도 비율 막대그래프

3. **Smart Score 한국판 — 종합 투자 점수**
   - TipRanks Smart Score (8요소)를 한국화:
     - 유튜버 컨센서스 + 공시 센티먼트 + 증권사 리포트 + 외국인/기관 수급 + 재무건전성
   - 1~10점으로 종목 종합 평가
   - **프리미엄 기능으로 과금 가능**

4. **인사이더 거래 트래커 한국판**
   - 한국은 임원 주식 변동 공시(DART)가 있음 → 이걸 자동 크롤링 + 시각화
   - "이 회사 대표이사가 최근 3개월 자사주 10억 매수" → 강력한 매수 시그널

#### 적용 Phase: Phase 1 (유튜버 수익률) + Phase 2 (컨센서스, 점수)

#### 한국 투자자 맥락:
- 한국 개인투자자의 40%+ 가 유튜브에서 투자 정보 획득 (한경 설문)
- 그러나 "이 유튜버 말 진짜 맞았나?"를 확인할 방법이 **전혀 없음**
- TipRanks 모델을 유튜버에 적용하면 → 한국 투자 시장의 **투명성 혁명**
- "유튜버도 성과가 추적된다" → 책임감 있는 추천 문화 형성

#### 경쟁사 대비 차별화:
- **알파스퀘어:** 종목 분석 + 차트 중심 → 유튜버 추적 없음
- **증플:** 유튜버 추천 리스트는 있으나 → 수익률 추적/랭킹 없음 (단순 나열)
- **우리:** 유튜버 추천 + 수익률 검증 + 랭킹 = **증플 + TipRanks 결합**

#### 수익모델 연결:
- **무료:** 유튜버 최신 픽 5개까지, 기본 수익률
- **프리미엄 (월 9,900원):** 전체 유튜버 랭킹, 수익률 상세, Smart Score, 알림
- **프로 (월 19,900원):** API, 포트폴리오 통합, 실시간 알림, 인사이더 트래커
- **예상 전환율:** 유사 서비스 기준 3~5%, MAU 10만 기준 월 3,000~5,000만원 잠재

#### 구현 난이도: 중
- 유튜버 픽 데이터는 이미 시그널 파이프라인에서 수집 중
- 수익률 계산 = 추천일 종가 기준 자동 계산 (주가 API 연동)
- 프론트엔드 시각화가 핵심 (차트, 랭킹 테이블)

#### 우선순위: P0

### 8. 총평
⭐️ 벤치마킹 가치: ★★★★★ (5/5)
📝 한줄 코멘트: **TipRanks의 "전문가 수익률 추적" 모델을 "유튜버 수익률 추적"으로 이식하는 것이 invest-sns의 핵심 차별화 전략. 이것 하나만 제대로 해도 한국 시장에서 독보적 포지션.**

## 🔬 투자 SaaS 해부 #002 — 알파스퀘어
📅 조사일: 2026-03-05
🏷 카테고리: [A] 종목 분석/데이터, [C] 소셜/커뮤니티
🌍 국가: 한국
🔗 URL: https://alphasquare.co.kr

### 1. 한줄 요약
차트·종목 분석·포트폴리오 관리·커뮤니티를 하나로 합친 한국의 올인원 스마트 트레이딩 플랫폼.

### 2. 타겟 고객
- **주 타겟:** 한국 개인투자자 (초급~고급), 특히 차트 분석 관심자
- **부 타겟:** 퀀트/시스템 트레이딩 관심자
- **고객 규모:** 누적 가입자 50만+ 추정, MAU 10~20만 추정 (앱스토어 다운로드 기반)

### 3. 핵심 기능 (최대 5개)
- **고급 차트:** TradingView급 커스텀 차트 도구 (한국 주식 특화). 보조지표, 그리기 도구, 멀티차트
- **종목 스크리너:** 재무/기술적 조건으로 종목 필터링. 퀀트 투자 지원
- **포트폴리오 관리:** 보유 종목 수익률 자동 추적, 자산배분 시각화
- **커뮤니티 (스퀘어):** 종목별 토론방, 아이디어 공유. 투자 아이디어 발행
- **알림:** 가격 알림, 거래량 급등 알림, 공시 알림

### 4. 수익모델
- **과금 방식:** 프리미엄 구독 (월/연)
- **가격대:**
  - Basic: 무료 (기본 차트, 제한된 스크리너)
  - Premium: 월 11,000원 / 연 99,000원
  - Professional: 월 33,000원 / 연 297,000원
- **기타:** 광고 수익 (무료 사용자)
- **ARPU 추정:** 유료 전환율 2~3%, ARPU 약 15,000원/월

### 5. 투자/성장
- **운영사:** 주식회사 파인랩스(FineLabs)
- **투자:** 시리즈A~B 추정 (THE VC 기준 누적 수십억 규모)
- **MAU:** 10~20만 추정
- **성장:** 2018년 출시 이후 꾸준한 성장, 한국 개인투자자 차트 플랫폼 중 상위권
- **앱스토어:** iOS/Android 양대 마켓 출시, 평점 4.0~4.5

### 6. 강점/약점
- 💪 **강점:**
  - 한국 주식 특화 차트가 매우 강력 (TradingView 한국판)
  - 스크리너 + 커뮤니티 + 포트폴리오 올인원
  - 모바일 앱 UX가 깔끔
  - 한국 투자자 커뮤니티가 활성화
- 😤 **약점:**
  - 유튜버/인플루언서 관련 기능 전무
  - 공시 분석/해석 기능이 약함 (단순 알림만)
  - 프리미엄 가격이 개인투자자에게 부담 (월 33,000원 프로)
  - 뉴스/리포트 큐레이션이 약함
  - AI 활용이 아직 초기 수준

### 7. invest-sns 적용 분석 ⭐️ (가장 중요)

#### 우리가 가져올 기능:
1. **종목 상세 페이지 구조**
   - 알파스퀘어의 종목 페이지 구성 참고: 기본정보 + 차트 + 재무 + 뉴스 + 커뮤니티
   - 우리 버전: 기본정보 + 유튜버 픽 + 공시 + 뉴스 + 증권사 리포트 + 커뮤니티
   - 알파스퀘어에 없는 "유튜버 픽" 레이어가 우리만의 차별화

2. **포트폴리오 연동 알림**
   - "내 포트폴리오 종목에 유튜버가 새 영상 올리면 알림"
   - 알파스퀘어는 가격 알림만 → 우리는 콘텐츠 알림까지
   - **프리미엄 핵심 기능**

3. **커뮤니티 구조 참고 (단, 차별화)**
   - 알파스퀘어 "스퀘어"는 일반 토론 → 우리는 "검증된 유튜버 의견" 중심
   - 누구나 글 쓰는 게시판 vs 검증된 전문가 의견 + 수익률 = 신뢰도 차이
   - Phase 3에서 커뮤니티 도입 시 참고

4. **스크리너 아이디어**
   - "유튜버 3명 이상이 추천한 종목" 필터
   - "최근 1주 공시 발생 + 유튜버 언급 종목" 필터
   - 기존 재무 스크리너에 우리만의 소셜 레이어 추가

#### 적용 Phase: Phase 1 (종목 페이지) + Phase 2 (알림) + Phase 3 (커뮤니티)

#### 한국 투자자 맥락:
- 알파스퀘어 사용자 = 차트 분석 좋아하는 중급+ 투자자
- 이들도 유튜브 투자 콘텐츠를 봄 → 그러나 알파스퀘어에서 유튜버 정보를 볼 수 없음
- 알파스퀘어 사용자가 "유튜버 픽 수익률"을 보려면 우리 사이트에 와야 함
- **알파스퀘어 보완재 포지션** → 직접 경쟁이 아닌 다른 축

#### 경쟁사 대비 차별화:
- **알파스퀘어:** 차트 + 데이터 강자 (기술적 분석 중심)
- **우리:** 콘텐츠 + 시그널 강자 (유튜버·공시·뉴스 중심)
- 겹치는 영역: 종목 상세 페이지 (그러나 우리만의 유튜버 레이어가 차별화)
- 알파스퀘어가 유튜버 추적 기능을 추가하기 전에 **시장 선점**이 핵심

#### 수익모델 연결:
- 알파스퀘어의 가격대(월 11,000~33,000원)를 벤치마크
- 우리는 더 낮은 진입장벽: 월 9,900원으로 유튜버 픽 + 공시 알림
- 알파스퀘어 유료 사용자 = 투자 정보에 돈 쓸 의향 있는 사람 → 우리 타겟과 겹침

#### 구현 난이도: 중
- 종목 상세 페이지는 이미 구조 설계 중
- 알림 시스템은 Phase 2에서 별도 구현 필요

#### 우선순위: P1

### 8. 총평
⭐️ 벤치마킹 가치: ★★★★☆ (4/5)
📝 한줄 코멘트: **한국 시장의 가장 가까운 경쟁자이자 참고 대상. 차트·데이터에선 이길 수 없으니 "유튜버 콘텐츠 + AI 분석"이라는 다른 축으로 차별화. 알파스퀘어가 유튜버 기능을 추가하면 위협이므로 선점 속도가 중요.**
