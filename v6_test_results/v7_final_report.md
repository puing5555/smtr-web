## V7 최종 개선안

### 🆕 새 규칙 (7개)

#### V7-R1: 유료 콘텐츠 티저 처리
```
"더 자세한 종목은 멤버십/아카데미에서", "유료 강의에서 공개"
→ 무료 영상에서 언급된 종목: mention_type = "티저"
→ 시그널 = 긍정 (최대), confidence = low
→ reasoning: "유료 콘텐츠 연계, 불완전 정보"
```

#### V7-R2: 전략/팩터/스타일 추천 처리
```
"소형 가치주", "모멘텀 전략", "배당주" 등 개별 종목이 아닌 투자 전략
→ market = STRATEGY (신규)
→ 시그널 테이블에서 제외 (요약에만 기록)
→ 단, "대표 종목"을 함께 언급하면 해당 종목만 시그널
예: "소형 가치주 전략 추천. 대표적으로 OO건설" → OO건설만 시그널
```

#### V7-R3: 스크리닝/필터 결과 처리
```
"이 조건으로 필터하면 N종목 나옵니다" (일괄 나열)
→ mention_type = "스크리닝"
→ 시그널 = 중립 (전체 리스트는 시그널 아님)
→ 단, 그 중 특정 종목을 "특히 주목"하면 해당 종목만 시그널
```

#### V7-R4: 자산배분/포트폴리오 조언 처리
```
"주식 60:채권 30:금 10" 또는 "적립식으로 매월 사세요"
→ 포트폴리오 전체를 1개 시그널로 취급
→ mention_type = "자산배분"
→ 시그널 테이블에서 제외 (요약에만 기록)
→ 개별 자산에 대한 명시적 의견이 있으면 그것만 시그널
```

#### V7-R5: 시리즈/연속 영상 처리
```
동일 주제 1부/2부/3부 등:
→ 최종편(마지막 편) 기준으로 1시그널
→ 이전 편은 시그널 제외
→ reasoning에 "N부작 시리즈 최종편 기준" 기록
```

#### V7-R6: 과거 시제/보유 보고 처리
```
"나는 이미 ~샀다", "내 포트에 ~있다" (과거/현재 보유)
→ mention_type = "보유"
→ 시그널 = 긍정 (보유 사실 ≠ 현재 매수 추천)
→ "지금도 추가 매수 중" → 매수 (현재 진행형)
```

#### V7-R7: 광고/PPL 감지
```
"오늘 OO와 함께", "협찬", "광고 포함" 등 광고 표기 감지 시:
→ confidence = low (무조건 하향)
→ reasoning에 "광고/협찬 가능성" 기록
→ 시그널 자체는 유지 (광고라도 의견은 의견)
```

### ✏️ 수정 규칙 (4개)

#### V7-M1: market 필드 확장 (7→9종)
```
기존: KR / US / CRYPTO / SECTOR / INDEX / ETF / MACRO
추가: REALESTATE / STRATEGY
→ REALESTATE: 아파트, 오피스텔, 토지, REITs
→ STRATEGY: 팩터, 스타일, 퀀트 전략
→ REALESTATE 시그널은 유지 (부동산도 투자 의견)
→ STRATEGY 시그널은 제외 (요약에만 기록)
```

#### V7-M2: mention_type 확장 (5→8종)
```
기존: 결론 / 논거 / 뉴스 / 리포트 / 교육
추가: 티저 / 보유 / 스크리닝
→ 티저: 유료 콘텐츠 연계 불완전 정보
→ 보유: 과거/현재 보유 보고
→ 스크리닝: 필터/조건 결과 일괄 나열
시그널 제외 대상: 교육, 스크리닝
시그널 제한 대상: 티저 (최대 긍정, confidence=low), 보유 (최대 긍정)
```

#### V7-M3: 라이브 영상 특별 규칙
```
라이브/실시간 방송 (2시간+ 영상):
→ "1영상 1종목 1시그널" 규칙 유지하되,
→ 같은 종목 매수→매도→매수 반복 시 최종 포지션 기준
→ 시청자 질문 응답 = 진행자 질문과 동일 취급 (게스트 규칙 적용)
→ 초단기 매매 실황 ("지금 들어가요") = 시그널 제외 (실시간 유효)
→ 라이브 중 요약/결론 발언만 시그널로 채택
```

#### V7-M4: 간접 인용 명확화
```
"전문가들은 ~라고 봅니다", "시장에서는 ~라는 의견이 있다"
→ speaker를 특정할 수 없는 간접 인용
→ mention_type = "뉴스", signal = "중립"
→ speaker = 채널명/앵커명 (실제 의견 주체 아님을 표시)
→ "전문가들"을 speaker로 쓰지 않음
```
