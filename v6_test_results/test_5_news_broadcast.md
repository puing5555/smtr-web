## V6 규칙 적용 결과

### 문제 패턴 발견

1. **간접 인용 문제**
   - "전문가들은 ~라고 봅니다" 
   - → 누구의 의견인지 불분명, 신호 제외 처리

2. **발언자 분리 어려움**  
   - 토론에서 여러 패널 동시 발언
   - → 타임스탬프로 개별 분리 필요

3. **뉴스 vs 코멘트 경계**
   - 앵커가 뉴스 전달하다 개인 의견 섞임
   - → 명확한 구분 기준 필요

4. **리포트 소개 코너**
   - 앵커가 증권사 리포트 내용 전달
   - → mention_type=리포트 적용 잘됨

### 잘 작동한 규칙

1. **게스트 발언만 시그널 추출** ✅
   - 진행자 질문은 제외, 게스트 답변만 추출

2. **시황 앵커 기본값** ✅  
   - 단순 시황 브리핑 = 뉴스/중립 처리

3. **"동의합니다" 제외** ✅
   - 독립적 시그널이 아닌 상대방 의견 동조는 제외

4. **리포트 규칙 적용** ✅
   - 증권사 리포트 인용 시 mention_type=리포트 정확히 분류

## 개선 권고사항

### 1. 간접 인용 처리 규칙 강화
```
"전문가들은/분석가들은/시장에서는" → 신호 제외
단, "ABC증권 김○○은" → 리포트 신호로 처리
```

### 2. 발언자 태깅 체계
```
[앵커] [게스트_A] [게스트_B] 등으로 화자 구분 후 
게스트 발언만 신호 추출
```

### 3. 뉴스/코멘트 구분 키워드
```
뉴스: "보도에 따르면", "발표했다", "나타났다"
코멘트: "전망입니다", "생각합니다", "예상됩니다"  
```
