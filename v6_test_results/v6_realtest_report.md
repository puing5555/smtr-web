## V7 반영 제안

### 신규 규칙 (실제 자막에서 검증됨)
1. **유료 티저 규칙**: "자세한 건 유료에서" 패턴 → 무료 부분만 분석, 시그널 강도 1단계 하향
2. **바스켓/섹터 규칙**: 섹터 단위 추천 → mention_type: "섹터추천", 개별 종목 시그널 아님
3. **전략 규칙**: 포트폴리오 배분 전략 → mention_type: "전략", 시그널에서 제외
4. **ETF 규칙**: ETF 추천 → INDEX ETF는 제외, 섹터 ETF는 시그널 가능
5. **카테고리 매도**: "반도체 빼고 현금화" → 매도 시그널 아닌 시장 전략으로 분류
6. **밈/유머 규칙**: 인터넷 밈/짤 인용 = 본인 의견 아님
7. **시그널 없음 출력**: 분석 후 시그널 0개일 때 "시그널 없음" 명시 + 영상 요약만 출력
8. **과거 보유 규칙**: "갖고 있었는데 팔았다" = 현재 시그널 아님
9. **복수 TP 합산**: 3개 이상 증권사 TP → 평균값 1시그널, mention_type: 컨센서스
10. **다종목 강도**: 깊은 분석(2분+) = 강한 시그널, 간단 언급 = 약한 시그널
