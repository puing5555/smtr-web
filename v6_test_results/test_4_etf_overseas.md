## 문제 패턴

### 1. 티커 vs 한글명 매칭 문제
**문제**: "테슬라"와 "TSLA" 매칭 실패
- 한글명으로 언급 → 종목 인식 실패 → 시그널 누락
- **심각도**: 높음 - 주요 시그널 누락 가능성

**예시**:
```
"테슬라가 급락했는데..." → 종목 인식 실패
"TSLA가 급락했는데..." → 정상 인식
```

### 2. INDEX 분류 경계선 문제
**문제**: S&P500, NASDAQ 언급 처리
- 지수 자체 언급 → INDEX로 올바르게 제외
- 하지만 "나스닥 종목들이 좋다" → 개별 종목 시그널로 오해석 위험

**예시**:
```
"S&P500이 상승세다" → INDEX (제외) ✓
"나스닥 종목들에 관심 가져야" → NASDAQ 시그널로 오해석 ✗
```

### 3. ETF 구성종목 전파 문제
**문제**: ETF 추천 → 구성종목 시그널 생성 금지 규칙 검증 필요
- QQQ 추천 → AAPL, MSFT 등 개별 시그널 생성되는지 확인 필요

### 4. 포트폴리오 비중 조절 분류
**문제**: "비중 줄이겠다" 류의 발언 처리
- 매도 시그널인가? 중립인가?
- confidence 수준은?

**예시**:
```
"테슬라 비중을 줄여야겠다" → SELL? CONCERN? 
"현금 비중을 늘리겠다" → 전체 시장 CONCERN?
```

### 5. 장기투자 vs 시그널 경계
**문제**: "적립식 투자" 조언의 시그널 여부
```
"VTI 적립식 투자 추천" → 교육인가? 시그널인가?
"장기 관점에서 애플은 좋다" → 전망인가? 권유인가?
```

### 6. 조건부 표현 처리
**문제**: confidence=medium 처리 기준 애매
```
"실적이 좋다면 매수" → medium (정상)
"아마도 상승할 것" → medium?
"확실히 매수 타이밍" → high?
```

## 잘 작동한 규칙

### 1. INDEX 시그널 제외
- S&P500, NASDAQ 지수 언급 → 올바르게 제외
- 개별 종목과 지수 구분 명확

### 2. MACRO 시장 분류

- 시그널 아닌 정보성 콘텐츠 구분
