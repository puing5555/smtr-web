## 🔧 V10.1 프롬프트 개선 내용

### 1. 규칙 30 강화: key_quote 품질 기준
**변경 전 (V10.0)**:
- 종목명 포함 + 투자 근거 + 15자 이상

**변경 후 (V10.1)**:
```
1. 종목명 명시적 포함 (약칭/대명사 금지)
   ❌ "이 회사", "둘 다", "여기"
   ✅ "삼성전자", "SK하이닉스", "TSMC"

2. 구체적 투자 판단 근거 (단순 감정 표현 금지)  
   ❌ "좋아요", "괜찮다", "추천합니다"
   ✅ "HBM4 수혜 확실", "영업이익 209% 증가"

3. 최소 20자 이상 (15자→20자 상향)

4. 완전한 문장 (단독으로 의미 파악 가능)
```

### 2. 규칙 29 강화: 화자 식별 우선순위
**변경 전**: 4가지 소스 나열

**변경 후**: **명확한 우선순위**
```
1순위: 영상 제목 "| 이름 직함" 패턴
2순위: 자막 내 명시적 게스트 소개  
3순위: 자막 내 호칭/직함 구분
4순위: 대화 패턴
```

### 3. 규칙 3 확장: 전망 vs 권유 패턴
**추가된 전망 표현**:
- "~해야 된다/되어야 한다" (당위성)
- "포트폴리오에 있어야" (소유 당위성)

**추가된 권유 표현**:
- "포지션을 넣고/늘리고" (구체적 행동)
- "분할매수/추가매수 하라" (구체적 방법)

### 4. 규칙 32 신규: 중복 방지 강화
```
- 바스켓 발언 중복 방지
- 같은 발언자 동일 내용 중복 방지  
- 목표주가 중복 방지
- video_id + speaker + stock 조합 중복 체크
```

### 5. 교차검증 강화: 26항목 → 29항목
V10.1 신규 검증 항목 3개 추가
