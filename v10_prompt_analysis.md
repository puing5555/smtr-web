### 1. key_quote 품질 기준 (53개 이슈 - 61%)
**현재 V10 규칙 30**:
- 종목명 포함 + 투자 판단 근거 + 15자 이상

**실제 DB에서 발견된 문제들**:
```
❌ "둘 다 사는게 좋습니다" (12자, 종목명 없음)
❌ "여전히 주인공이다라는 생각을 하고 있습니다" (23자, 종목명 없음)
❌ "자동차 회사가 로봇 회사로 바뀌면서 멀티플이 바뀌는 어마어마한 그런게 있다" (종목명 없음)
```

**근본 원인**: 규칙이 있지만 실행 시 체크가 안 됨. **약칭/대명사 허용** 기준이 모호.

### 2. 화자 식별 문제 (19개 이슈 - 22%)
**현재 V10 규칙 29**:
- 영상 제목 패턴, 호칭, 대화 패턴으로 게스트 식별

**실제 DB에서 발견된 문제들**:
```
❌ "고연수 하나증권 리서치센터 연구원" - 영상 제목에 명시된 게스트
❌ "박병창 교보증권 이사" - 영상 제목에 명시된 게스트  
❌ "권영화 교수" - 영상 제목에 명시된 게스트
```

**근본 원인**: 규칙이 있지만 **우선순위**가 모호. 영상 제목 > 자막 내 호칭 순서 불분명.

### 3. 전망 vs 권유 혼동 (2개 이슈 - 소수지만 중요)
```
❌ "포트폴리오에 당연히 있으셔야 된다" → 매수 (전망 표현인데 매수)
❌ "들어가야 된다... 포지션을 넣고 거기서 대응을 해야 된다" → 매수
```

**근본 원인**: **"~해야 된다" 패턴이 규칙3에 없음**. 권유인지 당위성 표현인지 애매.

### 개선 1: 규칙 30 key_quote 품질 기준 강화

**기존**:
```
1. 종목명 포함
2. 투자 판단 근거 포함  
3. 최소 15자 이상
```

**개선**:
```
1. **종목명 명시적 포함** (약칭/대명사 금지)
   ❌ "이 회사", "둘 다", "여기" 
   ✅ "삼성전자", "SK하이닉스", "TSMC"
   
2. **구체적 투자 근거 포함** (단순 감정 표현 금지)
   ❌ "좋아요", "괜찮다", "주인공이다"
   ✅ "HBM4 수혜 확실", "영업이익 209% 증가", "독점적 지위"
   
3. **최소 20자 이상** (15자→20자 상향)

4. **완전한 문장** (문맥상 완결된 문장)
   ❌ 앞뒤 맥락 없이는 의미 불분명한 문장 조각
   ✅ 단독으로 읽어도 투자 판단 이유를 알 수 있는 문장
```

### 개선 2: 규칙 29 화자 식별 우선순위 명시

**기존**:
```
자막에서 화자 구분이 안 될 때 추가 소스 활용:
1. 영상 제목 패턴...
2. 영상 설명란...
3. 자막 내 호칭...
4. 대화 패턴...
```

**개선**:
```
화자 식별 **우선순위** (상위가 우선):

1급. **영상 제목의 "| 이름 직함" 패턴**
   예: "| 권영화 교수 1부" → speaker: "권영화"
   
2급. **자막 내 명시적 소개**
   예: "오늘 모신 분은 박병창 교보증권 이사"
   
3급. **자막 내 호칭/직함**
   예: "교수님", "대표님", "박사님" 호칭 받는 발언자
   
4급. **대화 패턴**
   예: "~님 어떻게 보시나요?" 다음 발언자

⚠️ **검증 필수**: 채널명 = speaker일 때 → 게스트 영상인지 재확인
```

### 개선 3: 규칙 3 전망 vs 권유 패턴 확장

**기존**:
```
전망: ~이다/된다/보인다/전망이다/예상된다/가능하다/갈 수 있다/수밖에 없다
권유: ~해라/사라/담아라/들어가라/늘려라
```

**개선**:
```
전망 (→ 긍정):
- 기존 패턴 + **"~해야 된다/되어야 한다" (당위성 표현)**
- **"포트폴리오에 있어야"** (소유 당위성, 매수 액션 아님)
- **"~할 수밖에 없다"** (필연성 전망)

권유 (→ 매수):  
- **"지금이라도 사라/들어가라"** (명확한 타이밍 지시)
- **"포지션을 넣고/늘리고"** (구체적 행동 지시)
- **"분할매수/추가매수 하라"** (구체적 매수 방법 지시)

**애매한 경계선**: "~하는 게 좋겠다" → 문맥에 따라 긍정/매수 판단
```

## 🎯 수정된 V10.1 프롬프트 (핵심 부분)

### 수정된 규칙 30: key_quote 품질 기준
```
핵심 발언(key_quote)은 반드시 다음 4가지를 충족:

1. **종목명 명시적 포함**: 약칭/대명사 금지
   ❌ "이 회사", "둘 다", "여기", "그거"
   ✅ 정확한 종목명 또는 공식 약칭 (삼성전자, SK하이닉스, TSMC 등)

2. **구체적 투자 판단 근거**: 단순 감정 표현 금지  
   ❌ "좋아요", "괜찮다", "주인공이다", "추천합니다"
   ✅ "HBM4 수혜 확실", "영업이익 209% 증가", "독점적 지위", "메모리 사이클 전환"

3. **최소 20자 이상**: 맥락상 완결된 문장
   ❌ 15자 미만 또는 문맥 없이는 의미 불분명한 문장 조각
   ✅ 단독으로 읽어도 투자 판단 근거를 파악할 수 있는 완전한 문장

4. **투자 의견 명확성**: 발언자의 투자 입장이 명확히 드러남
   ❌ 단순 현상 설명, 뉴스 전달, 일반론
   ✅ "매수/긍정/경계" 등 투자 의견의 근거가 명확한 발언
```

⚠️ **필수 검증**: 채널명 = speaker일 때 → 게스트 영상인지 재확인 후 상위 우선순위 적용
